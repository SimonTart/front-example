IT之家8月23日消息 realme 印度 CEO 今日在推特放出了多款未上市的 realme 原型机，其中含有多种类型的背面设计，包括圆形后置镜头、矩阵后置镜头、横排和竖排摄像头设计等，还拥有独特的玻璃和配色设计。 本周三，Realme 表示本月将向印度市场投放两款入门级智能手机新品，分别为 C12 和 C15，将低于 10000 卢比（约合 929 元）。 由于印度市场需求较高，realme 在印度市场更倾向于采取所谓的 “机海战术”。IT之家了解到，目前 realme 在印度推出约 96 款机型，售价从 6499 至 47999 印度卢比（约合人民币 600 至 4432 元）不等。
